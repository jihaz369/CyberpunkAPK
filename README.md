# 📱 About

CyberpunkAPK is an open-source repository dedicated to Android application development.

The goal of this project is to provide modern Android applications with clean architecture, beautiful Cyberpunk UI concepts, and reusable components for developers.

Whether you are learning Android development or creating production-ready applications, this repository serves as a useful starting point.

# 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/jihaz369/CyberpunkAPK.git
```

Enter the project

```bash
cd CyberpunkAPK
```

Open using Android Studio.

Build

```bash
./gradlew assembleDebug
```

Release

```bash
./gradlew assembleRelease
```
